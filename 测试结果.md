## 📖 项目简介
本仓库是专为 **Drupal 11** 开发者准备的进阶指南。从零开始构建一个包含**多媒体交互、社交功能、以及支付集成**的现代化网站。

> **核心价值：** 解决国内开发者在 Drupal 主题开发（Twig + Bootstrap 5）以及云存储（GCS/S3）配置中的常见痛点。

## 🚀 核心课程模块

### 1️⃣ 主题开发 (Theming)
- [x] **Radix & Bootstrap 5**：从零构建响应式子主题。
- [x] **Twig 深度定制**：覆盖核心模板，实现复杂布局逻辑。
- [x] **Asset Management**：Drupal 11 下的库文件（Libraries）引用规范。

### 2️⃣ 功能集成 (Features)
- [ ] **多媒体中心**：利用 Media 模块构建图片与视频画廊。
- [ ] **支付系统**：Drupal Commerce 与国际支付接口集成。
- [ ] **S3 云存储**：基于 `s3fs` 的 GCS 静态资源托管最佳实践。

### 3️⃣ 效率工具 (AI Tools)
- [x] 使用 **Cursor / Trae** 进行代码自动补全与逻辑重构。
- [x] 编写自定义 **Drush** 命令提升维护效率。

## 💻 本地开发环境要求

| 组件 | 要求版本 | 备注 |
| :--- | :--- | :--- |
| **PHP** | `^8.3` | 需启用 `gd`, `opcache`, `pdo` 扩展 |
| **Database** | `MariaDB 10.6+` | 或 MySQL 8.0+ |
| **Web Server** | `Nginx` / `Apache` | 推荐使用 Docker DDEV 环境 |
